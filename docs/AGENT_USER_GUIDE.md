# AI Agent User Guide

## Introduction

Welcome to Prisma Glow's AI Agent system! Our specialized AI agents help you with tax, audit, accounting, and corporate services tasks.

## Getting Started

### Accessing Agents

1. Navigate to the **Agents** page from the main menu
2. Browse available agents by domain (Tax, Audit, Accounting, Corporate)
3. Use the search bar to find specific agents
4. Click on an agent to start a conversation

### Available Agents

#### Tax Agents
- **Malta Corporate Tax Specialist**: Malta CIT calculations, tax refunds, participation exemption
- **Rwanda Corporate Tax Specialist**: Rwanda CIT, EAC compliance, thin capitalization

#### Audit Agents
- **Audit Planning Specialist**: Materiality calculations, audit strategy development
- **Risk Assessment Specialist**: Inherent risk, control risk, fraud indicators

#### Accounting Agents
- **Bookkeeping Specialist**: Transaction classification, account reconciliation
- **Financial Reporting Specialist**: Financial statements, ratio analysis

#### Corporate Services Agents
- **Formation Specialist**: Entity type recommendations, formation documents
- **Governance Specialist**: Compliance deadlines, board minutes

## Using the Chat Interface

### Asking Questions

Be specific and provide context:

**Good**: "Calculate Malta tax refund for €100,000 trading income with €35,000 tax paid"

**Better**: "I have a Malta company with €100,000 trading income. We paid €35,000 in corporate tax. What refund can we claim when distributing dividends?"

### Understanding Responses

Agents will:
- Provide detailed calculations
- Show which tools were used
- Cite relevant regulations
- Offer recommendations

### Tool Calls

When agents use tools, you'll see:
- Tool name (e.g., `calculate_malta_tax_refund`)
- Input parameters
- Results

## Best Practices

### 1. Provide Complete Information
Include all relevant details:
- Amounts
- Dates
- Jurisdictions
- Entity types

### 2. Ask Follow-up Questions
Agents maintain conversation context:
```
You: "Calculate materiality for a company with €5M revenue"
Agent: [Provides calculation]
You: "What if profit is €500K?"
Agent: [Updates calculation with new information]
```

### 3. Verify Critical Information
Always verify agent responses with:
- Your tax advisor
- Auditor
- Legal counsel

## Limitations

### What Agents CAN Do
✅ Perform calculations
✅ Provide guidance based on rules
✅ Generate draft documents
✅ Identify compliance requirements

### What Agents CANNOT Do
❌ Provide legal advice
❌ File official documents
❌ Replace professional advisors
❌ Make final decisions

## Frequently Asked Questions

A: No. Agent responses are for informational purposes only. Consult qualified professionals for legal advice.
